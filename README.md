# forget-about-it

A spaced repetition system. Works, but still in Beta stages.

# How To

1. Run the provision executable _from a shell_. It outputs the login information, and if you just run it directly, it'll get lost.
2. Run the server executable from forget-about-it-$os-$arch. 
3. Go to http://localhost:8080 using a browser.
4. Enter the 12 words or the UUID.
5. Go to the overflow button and click Settings.
6. Click "add algorithm".
7. Click upload.
7. Click my-algorithm.json. You can find it online at https://github.com/forgetaboutitapp/my-algorithm/releases/tag/v0.0.03.
8. Click OK.
9. You should now see it added to the "General Settings" section.
10. Go Back
11. Go to the overflow window and click "Bulk Edit".
12. Add a question (for example, "|What is a house?|A building where people live|english translations").
13. Click Save
14. Go back.
15. check the box next to "english" and click "Quiz Me!"
16. You should see the questions!
